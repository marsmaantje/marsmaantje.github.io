Black Mesa Tram Ride

2021-present

The Black Mesa Tram Ride is an ongoing project of recreating the iconic Black Mesa tram ride in Resonite, a virtual reality metaverse that allows for real time collaborative development.
The project is being developed by a team of 2 people, including myself.
It started out as only the tram but has since evolved into importing the entire Black Mesa Research Facility into Resonite untill the Resonance Cascade.

For this project we are using the Black Mesa assets, a fan remake of the original Half-Life game in the Source engine.
But since Resonite is a completely different engine we had to recreate all systems to get everything functional.
